# Ultimate Finance Hackathon UI

This repository contains a high-fidelity, hackathon-ready UI scaffold built with Next.js, TypeScript, Tailwind CSS, Lucide icons and Recharts charts. The style is dark + glassmorphism with an emerald primary color.

Quick start

1. Install dependencies

2. Run dev server

Pro-tips
- Prioritize Data Density, Visual Cues, and Responsiveness
- Use component libraries (shadcn/ui, Mantine) to speed up polish
- Implement skeletons and empty states to avoid looking broken on first run

Files of interest
- `pages/index.tsx` — main dashboard layout
- `components/*` — modular UI pieces (Sidebar, Header, Charts, Transactions)
